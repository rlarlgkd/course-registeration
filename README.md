# course-registeration

학생들을 위한 수강신청 앱
백엔드 : Node JS
프론트 엔드 : HTML, CSS, JAVASCRIPT
템플릿 엔진 : NUNJUCKS
웹서버 : NGINX (예정)

### 사용법

```sh
$ npm install
```

```sh
$ npm start

# http://localhost:3000 로 들어가서 서버 켜졌는지 확인
```
